# Treasure Island Adventure Game 🏝️

An interactive text-based adventure game built with Python.

## Features
- ASCII art island display
- Turtle graphics animations
- Animated fireworks on win/lose
- Multi-path decision tree gameplay
- Multiple endings based on user choices

## Technologies Used
- Python
- Turtle module
- Random module

## How to Play
- Run the program
- Choose left or right at crossroad
- Wait or swim at the lake
- Choose a door - red, yellow or blue
- Find the treasure to win!
